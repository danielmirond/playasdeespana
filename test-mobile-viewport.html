To Test Mobile Viewport

In Safari/Chrome:
  1. Press Cmd+Option+U (Safari) or Cmd+Option+I (Chrome)
  2. Click the device icon or press Cmd+Shift+M
  3. Select "iPhone" (375px width)
  4. Refresh the page
Expected: Hamburger menu should appear in top-left corner

CSS Media Query Implementation

/* Mobile Navigation Menu */
.mobileNav {
  display: none;  /* Hidden by default */
}

/* Show on mobile screens */
@media (max-width: 767px) {
  .mobileNav {
    display: block;    /* Visible on mobile */
    position: relative;
  }
  /* ... hamburger, menu, animations ... */
}

/* Hide on desktop */
@media (min-width: 768px) {
  .mobileNav {
    display: none !important;  /* Ensure hidden */
  }
}
